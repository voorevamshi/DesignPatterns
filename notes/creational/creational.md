1. Creational Design Patterns

These focus on flexible object creation, decoupling systems from instantiation logic.

-   **[Singleton](singleton.md):**  Ensures a single instance.
-   **[Factory Method](factoryMethod.md) & [Abstract Factory](abstractFactory.md):**  Create objects via methods or families of related objects.
-   **[Builder](builder.md):**  Separates complex object construction.
-   **Prototype:**  Creates objects by cloning.
